# Анализ влияния опозданий доставки на маркетплейсе Olist

**Цель проекта:** показать, как опоздания доставки влияют на:
- пользовательский опыт (UX) — оценки и негативные отзывы
- денежные показатели (медианная стоимость заказа)

**Источник данных:**  
[Olist Brazilian E-Commerce dataset](https://www.kaggle.com/olistbr/brazilian-ecommerce)  
Данные за период 2016–2018 гг.

Данные необходимо скачать и положить в директорию проекта [./data](https://github.com/semenovapolinaa/Olist-analytics/tree/main/data)

## Основные метрики

- Средняя оценка заказа (UX)  
- Доля плохих отзывов (оценка ≤ 2)  
- Медианная стоимость заказа  
- Статус доставки (опоздание / вовремя)

---

## Ключевые выводы

1. Опоздание доставки ухудшает UX: средние оценки ниже, доля негативных отзывов выше.  
2. Опоздавшие заказы дороже медианно, что связано с логистической сложностью.  
3. Комбинация UX и денежных показателей показывает ключевые бизнес-риски.  
4. Массовые сбои видны на уровне месяца (пример: октябрь 2016).  
5. Различия между группами статистически значимы (p-value < 0.05).

---

## Бизнес-рекомендации

1. Сокращение количества опозданий через оптимизацию логистики.  
2. Приоритизация дорогих и сложных заказов.  
3. Работа с клиентским опытом при задержках (уведомления, компенсации).  
4. Мониторинг KPI: доля опоздавших заказов, median AOV, bad review rate.

---

## Ограничения анализа

- Данные исторические (2016–2018) — могут отличаться от текущей ситуации.   
- Влияние внешних факторов (сезонность, праздники, маркетинговые акции) не учитывалось.  
- Для визуализаций оценки были округлены, но статистический анализ проводился на исходных данных.  
- Некоторые данные смоделированы для тестирования гипотез.

---

## Потенциальный эффект внедрения рекомендаций

- Повышение UX → рост повторных покупок и удержания клиентов.  
- Снижение негативных отзывов → улучшение репутации маркетплейса.  
- Оптимизация денежного потока → снижение потерь на дорогих заказах.  
- Превентивная аналитика → меньше массовых сбоев, стабильная работа платформы.
